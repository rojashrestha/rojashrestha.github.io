Inventory Management System

A collaborative project by Saloni Rauniyar, Nischal Chaudhary, and Roja Shrestha, built to simplify warehouse operations and automate restocking decisions.

The Inventory Management System is designed to handle multiple warehouses, track product flow, and generate predictive insights using a forecasting model (ARIMA). It automatically notifies the warehouse manager when stock levels reach a critical point, ensuring timely restocking and minimizing human error.

Technologies: Django • Python • Tailwind CSS • HTML • CSS • SQLite
